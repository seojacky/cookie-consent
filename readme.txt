Simple and GDPR-compliant cookie consent banner for WordPress with tracking script blocking.

== Description ==

Cookie Consent Banner is a lightweight WordPress plugin that displays a customizable cookie consent banner to help your website comply with GDPR and other privacy regulations.

**Key Features:**

* **GDPR Compliant** - Automatically blocks tracking scripts until user consent
* **Customizable Design** - Light/dark themes with flexible positioning
* **Multilingual Support** - Built-in support for multiple languages
* **Tracking Control** - Blocks Google Analytics, Facebook Pixel, Yandex Metrica
* **Performance Optimized** - Minimal impact on site speed
* **Accessibility Ready** - Screen reader friendly and keyboard navigation

**Supported Tracking Scripts:**
* Google Analytics & Google Tag Manager
* Facebook Pixel
* Yandex.Metrica
* And more...

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/cookie-consent/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Cookie Consent to configure the banner
4. Customize the text, position, and appearance as needed

== Configuration ==

Navigate to **Settings > Cookie Consent** in your WordPress admin to configure:

* Banner text and button labels
* Privacy policy link (automatic or manual)
* Banner position (top, bottom, or modal)
* Theme selection (light, dark, or custom)
* GDPR compliance settings
* Cookie expiration periods

== Frequently Asked Questions ==

= Does this plugin block tracking scripts? =

Yes, the plugin automatically blocks Google Analytics, Facebook Pixel, Yandex.Metrica, and other tracking scripts until the user gives consent.

= Is it GDPR compliant? =

Yes, the plugin includes features like consent management, tracking script blocking, and Do Not Track header respect to help with GDPR compliance.

= Can I customize the banner appearance? =

Yes, you can choose from light/dark themes, customize positioning, and add custom CSS for complete control over the appearance.

= Does it support multiple languages? =

Yes, the plugin includes multilingual support using the format {:en}English{:}{:ru}Russian{:} in your text settings.
